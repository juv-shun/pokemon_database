# 本番環境へのデータ投入手順

## 前提条件

- CSV ファイルが生成済み（`output/` ディレクトリ）
- Supabase 本番プロジェクトが作成済み
- `psql` コマンドがインストール済み

## 1. 環境変数の設定

### .env ファイルの作成と編集

```bash
cp .env.example .env
```

`.env` を編集：

```bash
SUPABASE_PROJECT_REF=vphcabhyqxdljkyijccc
SUPABASE_DB_PASSWORD=your_actual_password_here
```

## 2. マイグレーションの適用

```bash
# 初回のみ
supabase link --project-ref your-project-ref
supabase db push
```

## 3. データ投入

```bash
./scripts/import_to_supabase.sh --remote
```

確認プロンプトで `yes` と入力して続行します。

## コマンドリファレンス

```bash
# ローカル環境
./scripts/import_to_supabase.sh [--local]

# 本番環境
./scripts/import_to_supabase.sh --remote
```
